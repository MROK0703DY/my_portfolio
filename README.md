# Портфолио веб-разработчика

Современное, адаптивное портфолио-сайт для демонстрации навыков и проектов веб-разработчика. Проект включает интерактивные элементы, анимации, виджеты и несколько тематических страниц.

## 🚀 Особенности

- **Полностью адаптивный дизайн** – корректное отображение на всех устройствах
- **Современный стек технологий** – HTML5, CSS3, JavaScript (ES6+)
- **Интерактивные элементы** – анимированные кнопки, выпадающие меню, виджеты
- **Несколько страниц** – главная, новости, примеры работ, виджеты
- **Оптимизированная производительность** – минимизированные ресурсы, плавные анимации
- **Темная/светлая тема** – настраиваемая цветовая схема через CSS-переменные

## 🛠 Технологии

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Стилизация**: CSS-переменные, Flexbox, Grid, анимации
- **Иконки**: Font Awesome 6.4.0
- **Шрифты**: Google Fonts (Inter, Poppins)
- **Дополнительно**: Python для утилит

## 🎨 Дизайн и интерфейс

### Ключевые компоненты

1. **Навигационная панель** – фиксированная, с плавной прокруткой
2. **Герой-секция** – представление разработчика с призывом к действию
3. **Секция навыков** – визуализация технических компетенций
4. **Портфолио проектов** – сетка проектов с фильтрацией
5. **Виджеты** – интерактивные элементы (часы, погода и др.)
6. **Футер** – контактная информация и социальные ссылки

## 📱 Адаптивность

Проект использует mobile-first подход с breakpoints:
- **Мобильные устройства**: до 768px
- **Планшеты**: 768px - 1024px
- **Десктопы**: от 1024px

## 🚀 Быстрый старт

### Локальный запуск

1. Клонируйте репозиторий:
   ```bash
   git clone <repository-url>
   cd my_portfolio
   ```

2. Откройте `index.html` в браузере:
   ```bash
   # Для Windows
   start index.html
   
   # Для macOS
   open index.html
   
   # Для Linux
   xdg-open index.html
   ```

### Запуск Python-скрипта

```bash
python portfolio_m_time_now.py
```

## 🔧 Настройка

### Конфигурация ссылок

Для управления социальными ссылками используйте файл `js/links.js`:
```javascript
const myLinks = {
    github: "https://github.com/yourusername",
    telegram: "https://t.me/yourusername",
    // ... другие ссылки
};
```

### Изменение изображений

Обновите пути в `config.json`:
```json
{
  "github": "img/2026/github.png",
  "gitlub": "img/2026/gitlub.png",
  // ... другие пути
}
```

## 🌐 Развертывание

### GitHub Pages

1. Перейдите в настройки репозитория на GitHub
2. В разделе "Pages" выберите ветку `main` и папку `/ (root)`
3. Сохраните изменения – ваш сайт будет доступен по адресу `https://username.github.io/repository-name`

### Альтернативные хостинги

- **Netlify**: Перетащите папку проекта в интерфейс Netlify
- **Vercel**: Импортируйте репозиторий и настройте сборку
- **Cloudflare Pages**: Подключите репозиторий и укажите корневую директорию
